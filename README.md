# Andres Campos | Profile Landing Page

This repository contains the source code for my personal profile landing page, built with Flask, Jinja2, Docker, and deployed via Google Cloud Run to my domain `www.lacamposm.com`.

## Technology Stack

* **Backend**: Python 3.11 with Flask and Gunicorn
* **Frontend**: HTML5, CSS3, Jinja2 Templates (incorporating premium aesthetic design)
* **Infrastructure**: Docker, Docker Compose, Google Cloud Run, Google Cloud Build

## Deploy

To deploy this application to Google Cloud Run, execute the `setup.sh` script, which will submit a build to Cloud Build and map your custom domain.

```bash
chmod +x setup.sh
./setup.sh
```
